# VirtualShop
Loja online usando:

- MicroServiços 
- Asp.Net MVC (Razor View) 
- .NET Core 6 
- AutoMapper 
- OAuth 2.0 
- IdentityServer 
- EF Core 


Features:
- API de produto
- API de carrinho
- API de desconto
- API de Autenticação e Autorização - IdentityServer
